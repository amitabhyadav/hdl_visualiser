## HDL Visualiser
A python GUI for visualizing your HDL code, show I/Os, internal signals and architecture functionalities.

Support for:
- VHDL
- Verilog

![demo_gui](docs/gui_demo_v0.1.png)

To run, simply execute.
```python
python3 main.py
```
